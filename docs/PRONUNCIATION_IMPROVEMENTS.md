# 发音优化说明

## 📢 问题反馈

小朋友反馈：**读音不够标准**

## 🔍 问题分析

### 原因
1. **音频文件缺失** - 原本设计的 `/audio/*.mp3` 文件不存在
2. **依赖 Web Speech API** - 浏览器内置的 TTS 引擎质量参差不齐
3. **音标符号直接读** - 用英文字母拼读音标不够准确

## ✅ 优化方案

### 已实施的优化（方案 3）

#### 1. 优化语音引擎选择
```javascript
// 优先级排序：
1. Google 美音 (Google US English) - 最自然 ✅
2. 其他美式英语
3. 英式英语
4. 任何英语
```

#### 2. 保持原有发音顺序
**逻辑：** 先读音标 → 停顿 → 读单词

**理由：**
- 符合学习逻辑：先学音标，再学单词
- 保持原有用户体验
- 小朋友已经习惯这个顺序

#### 3. 优化音标映射表
使用能准确发出该音标的字母组合：

```javascript
// 元音 - 长元音
'i:': 'ee',      // 字母组合 ee 发 /i:/ 音
'u:': 'oo',      // 字母组合 oo 发 /u:/ 音
'ɑ:': 'ar',      // 字母组合 ar 发 /ɑ:/ 音
'ɔ:': 'or',      // 字母组合 or 发 /ɔ:/ 音
'ɜ:': 'er',      // 字母组合 er 发 /ɜ:/ 音

// 元音 - 短元音
'ɪ': 'ih',       // 短 i 音
'ʊ': 'uh',       // 短 u 音
'ʌ': 'uh',       // 短 a 音
'ɒ': 'ah',       // 短 o 音
'e': 'eh',       // 短 e 音

// 元音 - 双元音
'eɪ': 'ay',      // 字母组合 ay
'aɪ': 'eye',     // 字母 i
'ɔɪ': 'oy',      // 字母组合 oy
'əʊ': 'oh',      // 字母 o
'aʊ': 'ow',      // 字母组合 ow

// 辅音
'θ': 'th',      // 清辅音 th
'ð': 'th',      // 浊辅音 th
'ʃ': 'sh',      // 字母组合 sh
'ʒ': 'zh',      // 字母组合 zh
'tʃ': 'ch',     // 字母组合 ch
'dʒ': 'j',      // 字母 j
'ŋ': 'ng',      // 字母组合 ng
```

#### 4. 优化语音参数
```javascript
// 音标发音：正常语速
rate: 1.0       // 音标用正常语速

// 单词发音：稍慢一点
rate: 0.85      // 单词稍慢，更清晰
```

## 🎯 使用建议

### 对于小朋友
1. **先听单词** - 认真听完整的单词发音
2. **再听音标** - 注意音标在单词中的发音
3. **跟读练习** - 大声跟读，录音对比

### 对于家长
1. **检查音量** - 确保设备音量适中
2. **安静环境** - 在安静环境下学习效果更好
3. **多次练习** - 每个音标至少听 3 遍

### 关键代码
```javascript
// 优化后的播放顺序
async speakPhoneticAndExample(symbol, exampleWord) {
  // 1. 先读单词（最标准）
  await this.speakWord(exampleWord)
  
  // 2. 暂停 500ms
  await new Promise(resolve => setTimeout(resolve, 500))
  
  // 3. 再读音标（用映射的单词）
  await this.speak(symbol, exampleWord)
}
```
