# 音标学习程序需求文档

**版本**: v1.0
**日期**: 2026-03-07
**状态**: 初稿

---

## 1. 项目概述

### 1.1 项目名称建议

| 名称 | 英文 | 含义 |
|------|------|------|
| 推荐 | Phonetic Fun | 音标 fun |
| 备选 | Sound Buddies | 声音伙伴 |
| 备选 | IPA Adventure | IPA 冒险 |

### 1.2 产品定位

一款面向 K-12 零基础英语学习者的**音标学习 H5 网页应用**，通过游戏化方式帮助非母语儿童掌握英语音标发音。

### 1.3 核心价值

- **有趣**：通过游戏化元素让学习音标变得有趣不枯燥
- **简单**：界面简洁，操作直观，适合儿童独立使用
- **可访问**：H5 网页，适配 iPad，随时随地学习
- **可监管**：轻量级家长监管功能

## 3. 技术架构

### 3.1 前端技术栈

| 技术 | 选择 | 理由 |
|------|------|------|
| 框架 | Vue 3 + Vite | 轻量、快速、适合 H5 |
| 状态管理 | Pinia | 轻量级状态管理 |
| 样式 | CSS3 + CSS Variables | 原生支持，易于定制主题 |
| 动画 | CSS Animation + GSAP | 简洁动画需求 |
| 构建工具 | Vite | 开发体验好，打包快 |

**替代方案**：React + Next.js 或 纯 HTML/CSS/JS（如果追求极致轻量）

### 3.2 音频方案

| 方案 | 实现方式 | 优点 |
|------|----------|------|
| 推荐 | 使用预录制 MP3 音频文件 | 音质稳定、兼容性好 |
| Web Speech API | 语音合成 | 无需额外音频文件，但发音质量不稳定 |

**音频文件建议**：
- 格式：MP3（兼容性最好）
- 采样率：44.1kHz
- 来源：可使用开源音标音频（如 Forvo、Dictionary 等）
- 存储：本地 `public/audio/` 目录

### 3.3 响应式设计（iPad 适配）

```
断点设计：
- 桌面：> 1024px
- iPad 横屏：768px - 1024px
- iPad 竖屏：< 768px
- 手机：< 480px
```

**适配要点**：
- 触摸友好：按钮最小 44px × 44px
- iPad Safe Area：考虑刘海屏和 Home 指示器
- 横竖屏适配：支持两种 orientation
- 视口单位：使用 vw/vh 实现流体布局

### 3.4 部署方案

| 方案 | 描述 | 适用场景 |
|------|------|----------|
| 静态托管 | Vercel / Netlify / GitHub Pages | 公开访问 |
| **本地部署** | **Nginx 静态部署** | **本地/内网使用** |

**推荐**：Nginx 本地部署（端口 8086 或 8087）

**Nginx 配置示例**：
```nginx
server {
    listen 8086;
    server_name localhost;
    
    root /home/vis/opencode/phonetic-fun/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 音频文件缓存
    location ~* \.(mp3|wav|ogg)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

## 4. 游戏化设计

### 4.1 适合儿童的游戏元素

#### 4.1.1 核心激励

| 元素 | 描述 |
|------|------|
| 星星系统 | 每次正确：+1 星；连续正确：+2 星 |
| 进度条 | 显示今日学习进度 |
| 徽章成就 | 收集成就徽章 |

#### 4.1.2 徽章设计

| 徽章 | 条件 |
|------|------|
| 🌟 初学者 | 完成第一个音标学习 |
| 🔤 音标小达 | 学习 10 个音标 |
| 🎯 元音大师 | 掌握全部 20 个元音 |
| 🔧 辅音高手 | 掌握全部 28 个辅音 |
| 🏆 全能选手 | 掌握全部 48 个音标 |
| 🎮 游戏达人 | 连续 5 次游戏正确 |

#### 4.1.3 小游戏：音标配对

```
流程：
1. 展示 4 个音标选项
2. 播放其中一个音标发音
3. 用户点击选择正确的音标
4. 正确：星星动画 + 下一题
5. 错误：提示正确音标 + 继续

难度设计：
- 简单：2 选 1
- 中等：4 选 1
- 困难：6 选 1
```

### 4.2 视觉风格

| 元素 | 建议 |
|------|------|
| 配色 | 明亮、柔和的儿童友好配色 |
| 字体 | 圆润、可爱的无衬线字体 |
| 图标 | 简洁的线性图标 |
| 动画 | 轻微的弹性动画，增加趣味 |

## 5. 内容结构

### 5.1 音标内容组织

```
音标分类：
├── 元音 (20个)
│   ├── 单元音 (12个)
│   │   ├── 前元音：/iː/ /ɪ/ /e/ /æ/
│   │   ├── 中元音：/ɜː/ /ə/ /ʌ/
│   │   └── 后元音：/uː/ /ʊ/ /ɔː/ /ɒ/ /ɑː/
│   └── 双元音 (8个)
│       ├── 合口双元音：/eɪ/ /aɪ/ /ɔɪ/ /əʊ/ /aʊ/
│       └── 集中双元音：/ɪə/ /eə/ /ʊə/
│
└── 辅音 (28个)
    ├── 爆破音：/p/ /b/ /t/ /d/ /k/ /g/
    ├── 摩擦音：/f/ /v/ /θ/ /ð/ /s/ /z/ /ʃ/ /ʒ/ /h/ /r/
    ├── 破擦音：/tʃ/ /dʒ/ /tr/ /dr/ /ts/ /dz/
    ├── 鼻音：/m/ /n/ /ŋ/
    ├── 舌侧音：/l/
    └── 半元音：/j/ /w/
```

### 5.2 单词库建议

每个音标配套 2-3 个简单单词：

| 音标 | 单词示例 |
|------|----------|
| /iː/ | see, tree, beach |
| /æ/ | cat, hat, bad |
| /θ/ | think, three, thank |
| /ŋ/ | sing, ring, long |

**单词来源建议**：
- 使用 K-12 常见词汇
- 优先选择简单、直观的单词
- 避免同音词干扰
